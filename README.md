# Comic Nest

A lightweight, static comic reader designed for smooth reading on the web.
It focuses on simplicity, fast loading, and a clean reading experience without unnecessary overhead..

-   Responsive UI for mobile and small devices
-   Canvas-based reader with keyboard navigation
-   Comics are client-side protected
-   Images encrypted and rendered on canvas to reduce scraping
